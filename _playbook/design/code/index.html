Front end code

Get started with the vets.gov front end design elements.

Framework

Vets.gov uses the open source Foundation framework. Components in use include:

  • Grid
  • Block Grid
  • Breadcrumbs
  • Alert Boxes
  • Labels
  • Type
  • Visibility

Download Foundation | Github

Other building blocks

  • Vets.gov style sheets are compiled using Sass (Syntactically awesome style sheets).
  • Front end static content is built using Jekyll, an open source static site generator.
